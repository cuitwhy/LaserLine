# LaserLine

激光线中心线识别

环境：OpenCV3.0、VS2015、Qt5.8

OpenCV安装位置D盘根目录

#核心算法

使用了本项目下一pdf文档中的方法，感谢作者！！！

#使用方法

1.点击LoadImg选择图片。

2.因本程序算法只能正确识别某一个方向（垂直或者水平）的激光线，此处我使用的是垂直方向。

所以如果你的图片中激光线是水平方向，请点击RotateImg将其旋转即可，如果已经是垂直方向则可忽略。

3.根据中间的部分二值化图像展示的效果可以设置不同的二值化阈值，辅助程序达到更好的识别效果，输入完成以后回车即可。

4.点击最右方按钮识别或者重新识别。
